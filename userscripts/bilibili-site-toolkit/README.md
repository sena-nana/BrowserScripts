# Bilibili Site Toolkit

Lightweight Bilibili layout cleanup migrated from Bilibili Evolved-style helper scripts.

## Target

- `https://www.bilibili.com/*`
- `https://t.bilibili.com/*`
- `https://space.bilibili.com/*`

## Features

- Hides common ad and activity surfaces.
- Keeps video side content scrollable.
- Reveals tag areas that are often clipped.

## Permissions

- `GM_addStyle`: cleanup styles.

## Reference Scripts

- `Bilibili Evolved (Preview).user.js`
- `Bilibili Evolved 强化辅助 (非 Bilibili Evolved 本体).user.js`

## Manual Verification

1. Build and install `dist/bilibili-site-toolkit.user.js`.
2. Open Bilibili home, video, dynamic, and space pages.
3. Confirm ad/activity clutter is reduced and main content remains usable.
