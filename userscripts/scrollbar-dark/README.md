# Scrollbar Dark

Low-risk global scrollbar style migrated from the dark scrollbar reference script.

## Target

- `http://*/*`
- `https://*/*`

Excludes selected Bilibili, Zhihu, and Baidu hosts where the reference script already avoided conflicts.

## Features

- Applies a thin dark scrollbar style.

## Permissions

- No privileged GM API.

## Reference Script

- `细滚动条(dark).user.js`

## Manual Verification

1. Build and install `dist/scrollbar-dark.user.js`.
2. Open a normal webpage and confirm scrollbar styling applies.
3. Open an excluded host and confirm the script does not run.
