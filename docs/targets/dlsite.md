# DLsite Userscript Notes

## Scope

`dlsite-voice-links` runs on DLsite, Ci-en media pages, and ASMR-related pages.

## Known URL Patterns

- `https://www.dlsite.com/*`
- `https://ci-en.dlsite.com/*`
- `https://media.ci-en.jp/*`
- `https://asmr.one/*`

## Required Permissions

- `GM_addStyle` for link and action styling.

## Debug Steps

1. Open a DLsite product page with an RJ code.
2. Confirm RJ/VJ text is linked to DLsite product URLs.
3. Confirm an ASMR link appears only when an RJ code is available.
